LMR (Learning Material Resource)

User visits LMR page —> Uploads study materials (PDF, PPT, DOCX formats supported) —> System processes documents: PDFs are chunked via RecursiveCharacterTextSplitter with metadata embedded using Ollama embeddings —> Embeddings stored in and retrieved from Chroma vector database —> AI processes content using Gemini Pro (with Ollama fallback) —> Generates comprehensive outputs including: —> 1) Content summary with key topics and important concepts —> 2) Key questions with answers and difficulty levels —> 3) Quiz/practice tests with multiple-choice questions —> 4) Last-minute recall notes with key points and quick facts —> 5) Neatly formatted downloadable PDF —> All outputs generated in selected language (22+ Indian languages supported) —> User can solve generated quizzes and analyze performance based on uploaded documents
